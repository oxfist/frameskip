# Frameskip
[![CircleCI](https://circleci.com/gh/oxfist/frameskip.svg?style=svg)](https://circleci.com/gh/oxfist/frameskip)

## Run project
Run project locally with Gradle by executing `./gradlew bootRun`.

## Run tests
Run tests by executing `./gradlew test`.
